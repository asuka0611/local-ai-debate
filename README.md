# ローカルAI討論システム

## 概要
複数のAIが異なる役割を持ち、1つのテーマについて議論するシステムを制作した。

従来のAIは単体で回答を出すことが多いが、本作品では
「賛成・慎重・批判」といった立場を持たせることで、
議論の過程そのものを可視化することを目的とした。

---

## 発想
AIの「結論」ではなく、「議論のプロセス」に価値を置いた点に新規性がある。

---

## 技術
- C言語で討論エンジンを実装
- Ollamaを用いたローカルAI連携
- HTTP通信の自作処理
- Node.jsとの連携
- Web UIとの統合

---

## デザイン
- チャット形式UI
- スマホアプリ風レイアウト
- 「考え中」演出によるリアルタイム感
